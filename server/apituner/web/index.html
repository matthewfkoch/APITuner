Recommended setup

Install the APITuner Agent APK, grant Display over other apps, Usage Access, and Notification Access, then add the device with backend http_agent on port 9092. Feature badges on each tuner card show what the backend supports; green means that feature is active on the device. On Fire Stick / Fire TV, use Grant permissions (ADB) on the tuner card (one-time network ADB) — those toggles are hidden in Fire Settings.
